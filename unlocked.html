Accès autorisé — Pièces déverrouillées

Vous avez déverrouillé le téléphone de la victime. Contenu simulé (exemples) :

  • Message vocal 1 : « Excuse-moi ma chérie, pardonne-moi, laisse-moi t'expliquer. »
  • Message du laboratoire : « Vos résultats sont arrivés, merci de passer les récupérer. »
  • Message amie : « Je commence à m'inquiéter, tu ne réponds plus… »

Note : dans le jeu, la fiche médicale (grossesse) peut être fournie séparément pour maintenir le suspense.
